# Abubakr Elmallah — Portfolio

**Personal Portfolio Website for Abubakr Elmallah | UC Irvine CS & iOS Developer**

A hand-built, dark-mode portfolio showcasing my apps, Discord bots, and high-school
projects. No frameworks, no templates — just HTML, CSS, and vanilla JavaScript.

🔗 **Live:** https://theabubakrabu.github.io/Abubakr-Elmallah-Website/

## Pages

- **Home** — intro, about, and contact
- **Projects** — the full catalog (UCI work, Apple ecosystem apps, Discord bots, high-school projects)
- **Star Wars** — Datapad, Aurebesh Translator, Sabacc & Aurebesh droids
- **Al-Islam** — Al-Islam, Al-Quran, and Al-Adhan
- **High School** — programming projects, custom flyers, and custom wallpapers

## Built with

- Plain **HTML / CSS / JavaScript** — zero dependencies, zero build step
- Custom cursor, magnetic links, scramble text, a flow-field canvas, and per-page launch animations
- Fully responsive and respects `prefers-reduced-motion`

## Structure

```
index.html          # landing page (root, for GitHub Pages)
assets/
  css/              # base · layout · components
  js/               # cursor · magnetic · scramble · reveal · clock · flowfield · intro · gallery
  img/              # me · apps · bots · awards · highschool · flyers · wallpapers
src/                # projects · star-wars · al-islam · high-school · resume
```

## Run locally

Just open `index.html` in a browser — no server or build needed.
